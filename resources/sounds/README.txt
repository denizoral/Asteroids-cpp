Sound Effects for Asteroids Game

Place the following sound files in this directory:

1. fire.wav - Sound when firing bullets
2. explosion.wav - Sound when destroying asteroids or player death
3. thrust.wav - Sound when player thrusts

You can find free sound effects from websites like:
- freesound.org
- opengameart.org
- soundbible.com

Recommended file format: 16-bit PCM WAV files
